# История изменений

## [2.0.0] - 2025-11-14

### Добавлено
- Выпадающий список для выбора смеси перед расчетом рациона
- Интеграция с облачной базой данных Supabase
- Автоматическая загрузка продуктов из базы данных при открытии страницы
- Синхронизация данных между устройствами через Supabase

### Изменено
- Переход с IndexedDB на Supabase для хранения данных
- Логика расчета рациона теперь использует выбранную смесь вместо автоматического подбора
- Упрощен интерфейс расчета рациона

### Технические изменения
- Добавлен файл `supabase-client.js` для работы с API Supabase
- Создана таблица `products` в Supabase с Row Level Security
- Обновлены функции `calculateDiet` и `generateDietWithProduct`
- Добавлена функция `loadProductsToSelect` для загрузки списка смесей

## Как использовать новый функционал

1. Откройте приложение
2. Заполните параметры пациента (вес, рост, возраст, пол, уровень активности)
3. **Выберите смесь из выпадающего списка "Выберите смесь для расчета"**
4. Нажмите кнопку "Рассчитать рацион"
5. Просмотрите результаты с рассчитанным количеством выбранной смеси

## Преимущества

- Более точный расчет рациона на основе конкретной выбранной смеси
- Облачное хранение данных позволяет использовать приложение с разных устройств
- Упрощенный интерфейс с понятным выбором продукта
- Автоматическое добавление начальной базы продуктов при первом запуске
